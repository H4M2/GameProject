Settings

  • Dark Mode This is the Dark Mode option. Once this setting is checked it will make the background dark grey.
  • Slow Mode This option will make the enemies move slower. Press Arrow Up to activate Slow mode.
  • Rocket Mode This option will make the game move very fast. Press Arrow Up to activate Rocket mode.

Run, Jump and Dive to Kill the Enemies!

and Hover over each setting to read their description/explanation!
